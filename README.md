# VRDAVis
## VRDAVis Backend
### How to build the server
#### Add submodules
```
git submodule update --init --recursuve
```
#### Build the backend
```
mkdir build
cd build
cmake ..
make
```
### How to run the server
```
./vrdavis_backend
```
## VRDAVis Frontend
### Install npm packages
```
npm install
```
### Run the React application
```
npm start
```
